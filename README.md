<h1>Student Course Registration System</h1>

<h3>Scenario</h3>
<p>This is the admin panel of a student and course registration system.Admins can register new Students, edit them and remove them. Same goes to the Courses.Main functions are enrolling students into courses and remove them.</p>

<p>In this pet project, i have implemented a simple REST Api using Spring Boot. As the Database System, i have used MySql.I did this project to refresh my knowledge on developing backends using Spring Boot.</p>
